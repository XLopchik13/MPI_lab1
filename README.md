# Лабораторная работа

1. Вычисление числа π методом Монте-Карло
2. Умножение матрицы на вектор (3 алгоритма: по строкам, столбцам и блокам)

### Запуск экспериментов
```powershell
cd python_analysis
python run_experiments.py
```

### Анализ и построение графиков
```powershell
python analyze_results.py
python plot_graphs.py
```

## Результаты

Скрипты Python построят графики:
- Время выполнения vs количество процессов
- Ускорение (speedup) vs количество процессов
- Эффективность vs количество процессов

Графики сохраняются в `results/`

## Параметры экспериментов

**Задача 1 (π):**
- Количество точек: 1,000,000 - 50,000,000
- Процессы: 1, 2, 4, 8

**Задача 2 (матрица × вектор):**
- Размеры матриц: 1000×1000, 3000×3000, 5000×5000
- Процессы: 1, 2, 4, 8
- Алгоритмы: по строкам, по столбцам, по блокам
